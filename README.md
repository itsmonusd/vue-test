# newvue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run command for fake json server installation
```
npm install -g json-server
```

### Run json-server
```
json-server db.json
```
